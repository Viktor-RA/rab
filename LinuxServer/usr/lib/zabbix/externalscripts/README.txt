Шаблон выполняет поиск датчиков только трёх типов: температуры, скорости вентиляторов и напряжения. Через модифицированный скрипт. Оригинальные комментарии условно актуальный.
Считывание значений из Windows 10 происходит непосредственно с WMI посредством удаленного агента.
Следует учитывать неприятный момент: при перезагрузке удаленной системы меняются ID датчиков и они переопределяются.
